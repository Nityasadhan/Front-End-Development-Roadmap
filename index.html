Frontend Development

Frontend Development is the Development of the graphical user interface of a webside, through the of HTML, CSS, and JavaScript , so that users can view and interact that webside.

  • HTML: The Hyper text Markup Language or HTML is the standard markup language for designed to be display in a web browser.

  • CSS: Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language as HTML or XML
  1. Pure CSS
  2. Bootstrap (Framework)
  3. Tailwind (Framework)
  4. Sass (Framework)
  • JavaScript: JavaScript often abbreviated js, is a programming language that is one of the core technologies of the World Wide Web, alongside HTML and CSS. It is use to add functionality in the website.
  1. Vanilla JS
  2. jQuery (Library)
  3. Rect (Framework)
  4. Angular (Framework)
  5. Vue (Framework)
  6. Typescript (Preprocessor)
